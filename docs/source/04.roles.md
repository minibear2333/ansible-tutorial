# 04.大项目管理

一个标准项目由多个`role`组成目录如下：

```BASH
roles/
│   └── webserver/
│       ├── tasks/
│       ├── templates/
│       ├── handlers/
│       ├── vars/
│       └── defaults/
│
└── playbook.yml
```

执行命令一次性创建成功

```BASH
mkdir -p {roles/,roles/webserver/,roles/webserver/tasks/,roles/webserver/templates/,roles/webserver/handlers/,roles/webserver/vars/,roles/webserver/defaults/}
```
